# Creating a collaborative session in Excalidraw 

`excalidraw` has been used in this project to create the whiteboard. A collaborative whiteboard is required for the project. To make the `excalidraw` whiteboard collaborative, it is planned to use the official portal of `excalidraw` given at `@excalidraw/utils`. 

## What is expected?
In the meeting, one of the members can create the whiteboard. The created whiteboard is shared with all the members in the meeting such that all the members can view the whiteboard. Then all the members of the meeting can draw and write on the whiteboard. The changes by any user must be reflected in the whiteboard of all the users.
