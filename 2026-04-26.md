# Changelog - 2026-04-26

## Summary
Whiteboard integration with tldraw in the meeting room. Split view: 80% whiteboard, 20% participants.

---

## Changes Made

### 1. Dependencies Added
- Installed `@tldraw/tldraw` and `@tldraw/sync` for real-time whiteboard collaboration

### 2. New Component: Whiteboard.jsx
- Created `src/components/Whiteboard.jsx`
- Uses tldraw SDK with useSyncDemo hook for collaboration
- Uses meetingId as room ID for shared whiteboard sessions
- Demo server provides 24-hour data persistence

### 3. Modified VideoTile.jsx
- Added `compact` prop for smaller display in participants panel
- Compact mode: reduced padding, smaller fonts, smaller border radius
- Non-compact mode: original styling

### 4. Modified MeetingPage.jsx
- Added import for Whiteboard component
- Added `whiteboardVisible` state (default: true)
- Split the main section into 80/20 layout:
  - 80%: Whiteboard (toggleable)
  - 20%: Participants panel with scrolling
- Passes whiteboard props to ControlBar
- Added compact prop to VideoTile components

### 5. Modified ControlBar.jsx
- Added whiteboard toggle button
- Shows "Hide Whiteboard" / "Show Whiteboard" based on state
- New props: onToggleWhiteboard, whiteboardVisible

---

## Features
- Real-time whiteboard collaboration (using tldraw sync demo server)
- Whiteboard visible by default
- Toggle whiteboard on/off via ControlBar
- Compact participants list with scrolling
- Shows participant count in header
